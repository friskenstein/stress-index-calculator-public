Stress Index Calculator

about

Johan Frisk 2021

×

To make a stress index calculator in the form of a web app had been on my mind for some time. It would be a nice complement to the stress index spreadsheet.
When I played around with a hexagonal grid idea one day I thought that it would be a cool layout for the stress index calculator. The end result is interesting, but I would rather use a more conventional layout myself. Admittedly, the hexagonal grid is not the most convenient layout. I still want to let this be a separate project and keep hosting it online. When I release a new design, I will add a link to it.
The background effect is certainly not resource optimized and could be too much for some devices. I might attempt to optimize it later or add a feature to turn it off. I'm not a designer, but I like nice graphics.
Please let me know if you have any suggestions or ideas. Any kind of feedback is welcome.
